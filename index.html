2. The Cipher System

Konju vocabulary is created by encoding English words using a Caesar cipher with +5 shift.

How It Works:

  1. Take an English word
  2. Shift each letter forward by 5 positions in the alphabet
  3. If you go past Z, wrap around to A
  4. Replace Q with KU (since Q isn't in Konju)
  5. Write the result using Konju characters

3. Grammar Structure

Word Order: SOV (Subject-Object-Verb)

Unlike English (SVO), Konju follows the pattern: Subject - Object - Verb

English: I eat food
Konju order: I food eat

Grammar Particles

Konju uses particles (suffixes) to mark grammatical relationships:

Particle Function Example
xeu Subject marker na-xeu = "I" (as subject)
y Object marker ktti-y = "food" (as object)
o Possessive marker na-o = "my"
a Location marker mtzhj-a = "at home"
eu Past tense jfy-eu = "ate"
yae Future tense jfy-yae = "will eat"
i Plural marker hfy-i = "cats"

4. Word Formation

Prefixes

Prefix Meaning Example
fe- opposite, not fe-ltti = bad (not-good)
re- again, repeat re-jfy = eat again

Suffixes

Suffix Meaning Example
-ka person who does jfy-ka = eater, chef
-lo thing related to jfy-lo = edible thing
-su place where jfy-su = restaurant
-zi adjective form jfy-zi = edible

Compound Words

Combine root words to create new meanings:

bfyjw (water) + mtzhj (home) = bfyjwmtzhj (bathroom)

7. Quick Reference

Sentence Pattern:

Subject-xeu Object-y Verb(-tense)

Key Particles:

  • xeu = subject marker
  • y = object marker
  • o = possessive
  • a = location
  • eu = past tense
  • yae = future tense
  • i = plural
